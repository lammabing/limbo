# Compare Crypto Providers Script

This script allows you to compare the outcomes generated by the different crypto providers (BCH, Bustadice, Stake) using the same client and server seeds.

## Usage

```bash
node cli-scripts/compare-providers.js <rounds> [clientSeed] [serverSeed]
```

### Parameters:
- `rounds`: Number of rounds to simulate
- `clientSeed` (optional): The client seed string
- `serverSeed` (optional): The server seed string

### Example:
```bash
node cli-scripts/compare-providers.js 10 "myClientSeed123" "myServerSeed456"
```

## Output

The script outputs a table showing the multiplier for each round across all three providers:

| Round | BCH Multiplier | Bustadice Multiplier | Stake Multiplier |
|-------|----------------|----------------------|------------------|
| 1     | 3.08x          | 4.55x                | 1.15x            |
| 2     | 1.17x          | 1.11x                | 2.44x            |

It also provides a summary with average multiplier, highest multiplier, and win count for each provider.
