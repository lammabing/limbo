How to Play

Manual Mode

  1. Enter your bet amount
  2. Set your target multiplier
  3. Click "Place Bet" to start the game
  4. The multiplier will increase until it crashes
  5. If the multiplier is above your target when it crashes, you win!

Auto Mode

  1. Configure your betting strategy
  2. Set stop conditions for profit and loss
  3. Configure bet progression (martingale)
  4. Click "Start Auto Bet" to begin
  5. The game will automatically place bets according to your strategy

Provably Fair

Our game uses a cryptographic system to ensure fairness. You can verify any game result using the client seed, server seed, and nonce.
